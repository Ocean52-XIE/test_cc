# 贪吃蛇游戏设计文档

## 项目概述

一个基于 HTML5 Canvas 的贪吃蛇游戏，采用关卡制设计，包含多种特殊食物和障碍物系统。

- **技术栈**: HTML5 + CSS3 + JavaScript (原生)
- **游戏类型**: 经典贪吃蛇 + 关卡挑战
- **仓库地址**: https://github.com/Ocean52-XIE/test_cc

---

## 游戏特性

### 关卡系统
- 共 **10 个关卡**，难度递增
- 每关有目标分数，达到即可过关
- 关卡越高，障碍物越多，速度越快

### 特殊食物系统

| 食物类型 | 颜色 | 效果 | 持续时间 |
|---------|------|------|---------|
| 普通食物 | 🔴 红色 | +10 分，蛇身变长 | - |
| 加速 | 🟢 绿色 | 移动速度加快 | 5 秒 |
| 减速 | 🔵 蓝色 | 移动速度减慢 | 5 秒 |
| 无敌 | 🟡 金色 | 穿墙、穿障碍、不怕撞自己 | 3 秒 |
| 缩短 | 🔴 浅红 | 蛇身缩短 3 节 | 即时 |
| 双倍分 | 🟣 紫色 | 吃普通食物得 20 分 | 8 秒 |

### 障碍物系统
- 从第 2 关开始出现障碍物
- 障碍物数量随关卡递增（2 → 20 个）
- 多种形状：方块、横条、竖条、L 形、T 形、十字形
- 无敌状态可穿越障碍物

## 核心逻辑

### 游戏循环
```
初始化 → 绘制 → 处理输入 → 更新状态 → 碰撞检测 → 绘制 → ...
```

### 碰撞检测
1. **撞墙**: 蛇头超出画布边界 → 游戏结束（无敌状态可穿墙）
2. **撞障碍物**: 蛇头碰到障碍物 → 游戏结束（无敌状态可穿越）
3. **撞自己**: 蛇头碰到蛇身 → 游戏结束（无敌状态无视）
4. **吃食物**: 蛇头与食物坐标重合 → 加分、生成新食物

### 数据持久化
- 最高分存储在 `localStorage.snakeHighScore`
